  1. ACCUEIL

    Créez votre premier site web :

    Apprenez les technologies les plus récentes pour devenir développeur web et créez des sites web avec HTML, CSS. Vous apprendrez aussi les bases de JavaScript pour rendre votre page interactive .

    2. A PROPOS

    • Introduction au développement web

      Faire la différence entre “Internet” et “Web”. Comprendre et résumer le fonctionnement du Web. Définir les API et les protocoles.

    • HTML

      Définir HTML, la structuration d’une page web, les tags. Créer des interfaces web avec les tags HTML

    • CSS

      Découvrir CSS. Travailler avec les sélecteurs CSS. Ajouter du design à votre site web avec CSS.

    • CSS layout

      Définir le positionnement avec CSS. Découvrir “Flexbox”, CSS, Grid, bootstrap. Définir le “responsive design” (site web reactif)

    • JavaScript

      Définir JavaScript. Apprendre les bases de JavaScript. Résoudre les problèmes avec JavaScript.

    • DOM

      Introduire DOM. Manipuler les structures HTML. Créer un site web dynamique.

    • Git et GitHub

      Définir Git et GitHub. Découvrir quelques fonctionnalités de GitHub. Manipuler les projets sur GitHub.
